Learn Coding using C++

Posted by CodingClass on December 6, 2023

Introduction

C++ is a fantastic tool for creating all sorts of things! It lets you build games, make art, explore data, and control robots. C++ is great for making the 'brains' or inner workings of programs and is really good at building things behind the scenes. With C++, you can even make your computer say 'hello' and create amazing programs. It's a fun way to learn coding and unlock your creativity!

Benefits

  1. Speedy Performance: C++ is super fast and efficient, perfect for games and complex programs.
  2. Useful Everywhere: It works for lots of things! From system software to high-level applications.
  3. Great Control: C++ gives lots of control over how things work, which is handy for certain types of programs.
  4. Makes Code Reusable: It helps organize code well, making it easier to reuse and build big things.

How To Get Started

Start coding in C++ with Dev-C++, an IDE made for C++ development. Download Dev-C++, create projects, write code, compile, and see outputs within the user-friendly interface. It simplifies learning by offering an intuitive workspace for beginners to code comfortably.
You can access Dev-C++ for download either through their official website or via this link: https://www.bloodshed.net/ then, you can follow the next steps:
On their website, scroll down until you spot 'Download original Dev-C++ 5,' as shown in the image below. Click it, and the file will automatically start downloading.
After finishing the application installation, let's create our first program. Before that, let's create a new file. Open the software, click on 'File' > 'New' > 'Source File...' or use Ctrl + Shift + N. Click 'OK,'
Then, we want to save our file and name your file with a .cpp extension (e.g., 'hello.cpp'). To do that, Click on ‘file’ > ‘Save as...’
In C++, we start by including necessary tools using '#include '. This allows us to display outputs. 'Using namespace std;' helps us print text like 'Hello, World!' easily. Then, 'int main() { return 0; }' is the fundamental setup of a C++ program.
Now we can finally start coding and create stuff. In this tutorial, we will learn how to print out 'Hello, World!'. To do this, we need to write 'cout <<', which means output in the C++ programming language. Then, we put 'Hello, World!' and a semicolon to end the line in C++.
